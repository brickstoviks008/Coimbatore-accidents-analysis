# 🚦 Coimbatore Road Accident Data Analysis

## 📌 Project Overview

Road traffic accidents are a major public safety concern in urban areas. This project focuses on analyzing road accident data from **Coimbatore** to identify patterns, trends, and factors contributing to accidents.

Using **Microsoft Excel, Power BI, and statistical analysis**, the project explores accident frequency, severity, time patterns, and possible hotspots. The insights generated from this analysis can help support **data-driven decision making for improving road safety**.

---

## 🎯 Project Objectives

* Analyze road accident data to identify **patterns and trends**
* Determine **high-risk time periods and accident hotspots**
* Study **accident severity distribution**
* Provide **visual insights using Power BI dashboards**
* Support **road safety awareness and planning**

---

## 🛠️ Tools & Technologies

* **Microsoft Excel** – Data cleaning and preprocessing
* **Power BI** – Dashboard creation and visualization
* **Statistics** – Data analysis and interpretation

---

## 📊 Data Analysis Process

### 1. Data Collection

Accident data related to **road incidents in Coimbatore** was collected and organized for analysis directly from the coimbatore police station.

### 2. Data Cleaning

The dataset was cleaned using **Excel** to:

* Remove missing or inconsistent values
* Standardize formats
* Prepare the data for analysis

### 3. Exploratory Data Analysis (EDA)

Key statistical insights were generated to understand:

* Accident frequency
* Severity levels
* Time and date patterns
* Location-based trends

### 4. Data Visualization

Interactive dashboards were created using **Power BI** to visually represent:

* Accident distribution
* Monthly and yearly trends
* Severity categories
* Key contributing factors

---

## 📈 Dashboard Insights

The Power BI dashboard highlights several important insights:

* Identification of **high-risk accident periods**
* Distribution of accidents by **severity level**
* Patterns in accidents across **different time intervals**
* Trends that may help identify **possible accident hotspots**

These insights can assist in **improving road safety measures and awareness**.

## 💡 Suggestions and Recommendations

Based on the analysis, several measures can be suggested to improve road safety in Coimbatore:

## 1. Improve Traffic Management

Implement better traffic signal coordination in busy intersections.

Increase traffic police presence during peak hours.

## 2. Identify and Address Accident Hotspots

Conduct detailed studies in high-risk locations.

Install warning signs, speed breakers, and better road markings in these areas.

## 3. Enhance Road Infrastructure

Improve street lighting, especially in areas with high night-time accidents.

Maintain road quality by repairing potholes and improving lane markings.

## 4. Promote Road Safety Awareness

Conduct public awareness campaigns on safe driving practices.

Encourage the use of helmets, seat belts, and speed control.

## 5. Use Data-Driven Policy Planning

Authorities can use data analytics and dashboard insights to monitor accident trends.

Regularly update accident datasets to enable continuous monitoring and policy improvement.
